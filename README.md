# Retail Sales Dashboard (Excel)

Interactive Excel dashboard built using Pivot Tables, Slicers, and Charts.

## 📊 Features
- Monthly Revenue Trend
- Revenue by Country (Top 5)
- Total Revenue KPI
- Total Orders KPI
- Total Customers KPI
- Year-based interactive slicer (2010–2011)

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Data Cleaning
- KPI Linking via Cell References

## 📁 Dataset
Online Retail dataset (2010–2011)

## 🎯 Objective
To analyze retail sales performance and build an interactive business dashboard for insights.
